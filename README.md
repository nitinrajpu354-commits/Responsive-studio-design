# Responsive Studio Design

A modern and fully responsive studio website built using HTML and CSS. The project focuses on creating a clean, visually appealing, and user-friendly interface that adapts seamlessly across different screen sizes.

## 🚀 Features

- Fully responsive design
- Clean and modern UI
- Mobile-friendly layout
- Smooth and consistent user experience
- Cross-device compatibility

## 🛠️ Technologies Used

- HTML5
- CSS3

- CSS Flexbox
- CSS Grid
- Media Queries
- Building mobile-friendly layouts
